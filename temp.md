**Day 3: Machine Learning (ML)**

### Main Content

* **Unsupervised Learning:** Unsupervised learning is a type of ML that is used when the data is not labeled, such as clustering customers based on their buying behavior.

### Definitions

* **Machine Learning (ML):** A type of AI that is designed to learn from data and make predictions or decisions.
* **Supervised Learning:** A type of ML that is used when the data is labeled.
